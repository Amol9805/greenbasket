GreenBasket is a front-end vegetable e-commerce website built using HTML.
The project simulates a real-world online grocery platform where users can browse vegetables by category, view discounted prices, and access login and sign-up pages.

This project focuses on UI structure, layout design, and multi-page navigation, making it suitable for learning front-end fundamentals and showcasing practical web development skills.

## Key Highlights:

- Category-based vegetable listings

- Discounted pricing with original price strike-through

- Clean navigation header with icons

- Login and Sign-Up authentication UI

- Informative footer with contact & payment details

## Technologies Used:

- HTML5

- Inline styling (no external CSS or JavaScript)
